# Конструктор учебных заданий
[Загрузить программу](https://github.com/ilyavasilev14/test_generator/releases)

[Репозиторий (программа, позволяющая удалённо хранить и загружать генераторы заданий)](https://github.com/ilyavasilev14/test_generator_repo)

Программа позволяет учителям быстро создавать различные виды работ с помощью готовых генераторов заданий, а также создавать
свои генераторы благодаря поддержке скриптинга на языке программирования Lua.

Программа была протестирована на:
- Windows 10 и 11
- Arch Linux
- Rosa Linux (Chrome Desktop 2021.1)
